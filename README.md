# Express Todo 
This is a simple express server for a todo list. The main purpose of the repo is to help get more comfortable with Express. There are three front-end versions written: Vanilla js, ReactJS, and react-native for iOS.

## Installation
1) Navigate into backend `$ cd backend`
2) In the terminal, execute `$ npm install` to install all the dependencies
3) Navigate to the frontend and open index.js
